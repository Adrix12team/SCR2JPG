# **TROUBLESHOOTING**

### **Q:** Why don't my screenshots appear in the 3DS Camera, even though they were converted successfully?

**A:** Most likely, this is the management file not being updated.

### **How to fix:**

1. Open the 3DS camera.
2. Navigate to to the settings, then press on **Data Management**.

<img width="1000" height="480" alt="step12" src="https://github.com/user-attachments/assets/fbad14d7-9787-43d0-b6da-aab41ff09234" />

3. You will now see the following:

<img width="400" height="480" alt="step3" src="https://github.com/user-attachments/assets/adb1d0e9-c859-4c36-8608-dfd9b4f36862" />

We can **safely ignore this**. Press **OK**.

4. After confirming, you will see this next:

<img width="400" height="480" alt="step4" src="https://github.com/user-attachments/assets/cf928f21-ce18-4912-b69a-7eb1a15921de" />

Press **No**. Now go back and exit the settings. **If you see the following, it worked.** If not, repeat the process:
